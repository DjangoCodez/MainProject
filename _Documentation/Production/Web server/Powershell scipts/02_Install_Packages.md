# 02_Install_Packages.ps1

## Purpose

This script is used to install necessary software packages such as Dacpac, WebDeploy, and .NET 9 Web Hosting.

It also installs SoftOne Status into c:\Status and adds Event source to Eventlog for SoftOne Status.

## Parameters

- **config.json**: The configuration file containing the installation paths.

## Usage

```powershell
./02_Install_Packages.ps1
```
